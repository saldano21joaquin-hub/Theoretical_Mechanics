¿Qué problema estamos tratando de resolver?

  • En mecánica elemental uno suele partir desde una ecuación diferencial:
  • $$m\ddot x = F$$
  • o, en varias dimensiones,
  • $$m\ddot{\vec r}=\vec F$$
  • Aquí vamos a invertir el punto de partida.
  • En lugar de preguntar primero por la fuerza y después por la trayectoria, vamos a preguntar:
  • entre todas las curvas posibles que unen dos puntos, ¿cuál es la curva físicamente o geométricamente privilegiada?
  • Ésa es la entrada al cálculo variacional.

¿Qué significa extremizar?

  • En cálculo ordinario, si tenemos una función $f(a)$ y queremos buscar un extremo, estudiamos cómo cambia al deformar $a$:
  • $$a \to a+\varepsilon$$
  • Por ejemplo, si $f(a)=a^2$ entonces
  • $$f(a+\varepsilon)=(a+\varepsilon)^2=a^2+2a\varepsilon+\varepsilon^2$$
  • El cambio es
  • $$f(a+\varepsilon)-f(a)=2a\varepsilon+\varepsilon^2$$
  • Si $\varepsilon$ es pequeño, el término relevante a primer orden es $2a\varepsilon$.
  • Para que $a$ sea un punto extremo, el cambio lineal debe anularse.
  • Entonces: $2a=0$ y se obtiene $a=0$.
  • La idea que nos interesa conservar es: extremizar significa hacer cero el cambio de primer orden.

De una variable a una curva completa

  • En cálculo variacional ya no se deforma un número aislado.
  • Se deforma una función completa.
  • Es decir, ya no buscamos un valor especial de $a$, sino una curva especial $y(x)$.
  • A cada curva $y(x)$ le asociamos un número que será una integral.
  • Una regla que toma una función y devuelve un número se llama funcional.
  • Una funcional típica tiene la forma:
  • $$I[y]=\int_a^b \mathcal E(y,y',x)\,dx$$
  • donde: $y(x)$ es la curva, $y'=\dfrac{dy}{dx}$, $\mathcal E$ es el integrando, e $I[y]$ es el número asociado a esa curva.

Primer ejemplo de funcional

  • Si queremos la curva más corta entre dos puntos del plano, usamos el elemento de arco:
  • $$ds=\sqrt{dx^2+dy^2}$$
  • Si la curva está escrita como $y(x)$, entonces $dy=y' dx$, luego
  • $$ds=\sqrt{dx^2+(y' dx)^2}=\sqrt{1+(y')^2}\,dx$$
  • Por tanto, la longitud total es:
  • $$I[y]=\int_a^b \sqrt{1+(y')^2}\,dx$$
  • Aquí el integrando es $\mathcal E(y,y',x)=\sqrt{1+(y')^2}$.
  • Lo importante es la estructura: a cada curva $y(x)$ le asignamos un número $I[y]$.

¿Cómo se varía una curva?

  • Si en cálculo ordinario hacíamos $a\to a+\varepsilon$, ahora hacemos lo análogo con una curva:
  • $$Y(x)=y(x)+\varepsilon \eta(x)$$
  • Aquí: $y(x)$ es la curva base, $\eta(x)$ es una deformación arbitraria, $\varepsilon$ es un parámetro pequeño, $Y(x)$ es una curva vecina.
  • Derivando:
  • $$Y'(x)=y'(x)+\varepsilon \eta'(x)$$
  • Si los extremos están fijos, imponemos $Y(a)=y(a)$ y $Y(b)=y(b)$.
  • Entonces:
  • $$\eta(a)=0,\qquad \eta(b)=0$$
  • Esto será esencial cuando integremos por partes.

Variación de la funcional

  • Evaluamos la funcional en la curva deformada:
  • $$I[Y]=\int_a^b \mathcal E(Y,Y',x)\,dx$$
  • Sustituyendo $Y=y+\varepsilon\eta$ y $Y'=y'+\varepsilon\eta'$, obtenemos:
  • $$I[Y]=\int_a^b \mathcal E(y+\varepsilon\eta,\; y'+\varepsilon\eta',\; x)\,dx$$
  • Ahora expandimos el integrando a primer orden en $\varepsilon$. Usando Taylor:
  • $$\mathcal E(y+\varepsilon\eta,\; y'+\varepsilon\eta',\; x)=\mathcal E(y,y',x)+\frac{\partial \mathcal E}{\partial y}\,\varepsilon\eta+\frac{\partial \mathcal E}{\partial y'}\,\varepsilon\eta'+O(\varepsilon^2)$$
  • Entonces:
  • $$I[Y]=\int_a^b \left[\mathcal E(y,y',x)+\varepsilon\frac{\partial \mathcal E}{\partial y}\eta+\varepsilon\frac{\partial \mathcal E}{\partial y'}\eta'+O(\varepsilon^2)\right]dx$$

Separación de términos

Expansión paso a paso
$$I[Y]=\int_a^b \mathcal E(y,y',x)\,dx$$
$$+ \varepsilon \int_a^b \frac{\partial \mathcal E}{\partial y}\eta\,dx$$
$$+ \varepsilon \int_a^b \frac{\partial \mathcal E}{\partial y'}\eta'\,dx$$
$$+ \int_a^b O(\varepsilon^2)\,dx$$
Agrupación y definición
$$I[Y]=I[y]+\varepsilon \int_a^b \left(\frac{\partial \mathcal E}{\partial y}\eta+\frac{\partial \mathcal E}{\partial y'}\eta'\right)dx$$
$$+O(\varepsilon^2)$$
Definimos la primera variación por:
$$\delta I=\int_a^b \left(\frac{\partial \mathcal E}{\partial y}\eta+\frac{\partial \mathcal E}{\partial y'}\eta'\right)dx$$

Condición de estacionariedad

  • El análogo de "anular el término lineal" ahora es imponer:
  • $$\delta I=0$$
  • Entonces la curva buscada debe satisfacer:
  • $$\int_a^b \left(\frac{\partial \mathcal E}{\partial y}\eta+\frac{\partial \mathcal E}{\partial y'}\eta'\right)dx=0$$
  • Pero todavía hay un problema: aparece $\eta'$.
  • Queremos reescribir todo en función de $\eta$ sola.
  • Para eso se integra por partes.

Integración por partes

Configuración de IPP
Consideremos:
$$\int_a^b \frac{\partial \mathcal E}{\partial y'}\eta'\,dx$$
Elegimos:
$$u=\frac{\partial \mathcal E}{\partial y'}$$
$$dv=\eta' dx$$
Entonces:
$$du=\frac{d}{dx}\left(\frac{\partial \mathcal E}{\partial y'}\right)dx$$
$$v=\eta$$
Aplicación de IPP
Por integración por partes:
$$\int_a^b \frac{\partial \mathcal E}{\partial y'}\eta'\,dx$$
$$=\left[\frac{\partial \mathcal E}{\partial y'}\eta\right]_a^b-\int_a^b \frac{d}{dx}\left(\frac{\partial \mathcal E}{\partial y'}\right)\eta\,dx$$
Como $\eta(a)=\eta(b)=0$, el término de borde se anula.
Luego:
$$\int_a^b \frac{\partial \mathcal E}{\partial y'}\eta'\,dx$$
$$=-\int_a^b \frac{d}{dx}\left(\frac{\partial \mathcal E}{\partial y'}\right)\eta\,dx$$

Reunión de términos

  • Sustituyendo en $\delta I$:
  • $$\delta I=\int_a^b \frac{\partial \mathcal E}{\partial y}\eta\,dx-\int_a^b \frac{d}{dx}\left(\frac{\partial \mathcal E}{\partial y'}\right)\eta\,dx$$
  • Reuniendo términos:
  • $$\delta I=\int_a^b \left[\frac{\partial \mathcal E}{\partial y}-\frac{d}{dx}\left(\frac{\partial \mathcal E}{\partial y'}\right)\right]\eta\,dx$$

Paso decisivo

  • La función $\eta(x)$ es arbitraria en el interior del intervalo.
  • Solo está fijada en los extremos.
  • Por tanto, para que:
  • $$\delta I=\int_a^b \left[\frac{\partial \mathcal E}{\partial y}-\frac{d}{dx}\left(\frac{\partial \mathcal E}{\partial y'}\right)\right]\eta\,dx=0$$
  • se cumpla para toda variación admisible $\eta(x)$,
  • el coeficiente de $\eta$ debe anularse punto a punto.
  • Así obtenemos la ecuación de Euler–Lagrange:
  • $$\frac{d}{dx}\left(\frac{\partial \mathcal E}{\partial y'}\right)-\frac{\partial \mathcal E}{\partial y}=0$$

Qué se acaba de demostrar

  • El cálculo variacional parte desde una condición global sobre curvas: hacer estacionaria una integral.
  • Y termina produciendo una ecuación diferencial local.
  • Ésa es la idea central del bloque.
  • Se puede resumir así:
  • curva $y(x)$
  • $\to$ deformación $Y(x)=y(x)+\varepsilon\eta(x)$
  • $\to$ cálculo de $\delta I$
  • $\to$ integración por partes
  • $\to$ ecuación de Euler–Lagrange

Ejemplo mínimo: la recta

  • Si $I[y]=\int_a^b \sqrt{1+(y')^2}\,dx$, entonces $\mathcal E=\sqrt{1+(y')^2}$.
  • Por tanto:
  • $$\dfrac{\partial \mathcal E}{\partial y}=0$$
  • y
  • $$\dfrac{\partial \mathcal E}{\partial y'}=\dfrac{y'}{\sqrt{1+(y')^2}}$$
  • Euler–Lagrange da:
  • $$\dfrac{d}{dx}\left(\dfrac{y'}{\sqrt{1+(y')^2}}\right)=0$$
  • Luego:
  • $$\dfrac{y'}{\sqrt{1+(y')^2}}=C$$
  • De aquí se sigue que $y'=\text{constante}$, luego:
  • $$y=mx+n$$
  • Es decir, la curva extremal es una recta.

Principio de Fermat

  • En óptica geométrica, la luz no se describe primero con fuerzas.
  • Se describe variacionalmente.
  • El tiempo de tránsito es: $t=\int dt=\int \frac{ds}{v}$
  • Si el medio tiene índice de refracción $n$, entonces $v=\dfrac{c}{n}$.
  • Por tanto: $dt=\dfrac{n\,ds}{c}$
  • Luego: $t=\dfrac{1}{c}\int n\,ds$
  • Como $1/c$ es constante, la funcional efectiva es:
  • $$I=\int n\,ds$$
  • Otra vez aparece exactamente la misma estructura variacional.

Paso a mecánica: principio de Hamilton

  • Ahora se cambia el problema geométrico por uno mecánico.
  • En lugar de una curva espacial $y(x)$, consideramos una trayectoria dinámica $q(t)$.
  • En lugar de una funcional genérica: $I[y]=\int \mathcal E(y,y',x)\,dx$,
  • consideramos la acción:
  • $$S[q]=\int_{t_1}^{t_2}L(q,\dot q,t)\,dt$$
  • Aquí $L$ se llama lagrangiano.

¿Qué es el lagrangiano?

  • El lagrangiano es la función que caracteriza dinámicamente al sistema.
  • En general depende de:
  • las coordenadas generalizadas $q_i$,
  • las velocidades generalizadas $\dot q_i$,
  • y eventualmente del tiempo $t$:
  • $$L=L(q_1,\dots,q_n,\dot q_1,\dots,\dot q_n,t)$$
  • Para sistemas conservativos, la forma más habitual es:
  • $$L=T-U$$
  • donde: $T$ es la energía cinética, $U$ es la energía potencial.
  • Esta forma no se inventa al azar: funciona porque conduce a ecuaciones equivalentes a Newton cuando el sistema es conservativo.

Mecánica Intermedia 2026

Qué postula el principio de Hamilton

  • Entre los tiempos fijos $t_1$ y $t_2$, el sistema pasa por configuraciones fijas en los extremos: $q(t_1)$ y $q(t_2)$.
  • El principio de Hamilton afirma que la trayectoria real es aquella que hace estacionaria la acción:
  • $$\delta \int_{t_1}^{t_2}L(q,\dot q,t)\,dt=0$$
  • La variación se hace con extremos fijos:
  • $$\delta q(t_1)=0,\qquad \delta q(t_2)=0$$
  • La estructura matemática es la misma que en cálculo variacional ordinario.
  • Solo cambian los nombres: $x\to t$, $y\to q$, $y'\to \dot q$, $\mathcal E \to L$.

Aplicación de la variación a la acción

  • Partimos de: $S[q]=\int_{t_1}^{t_2}L(q,\dot q,t)\,dt$
  • y variamos: $q(t)\to q(t)+\varepsilon \eta(t)$ con $\eta(t_1)=0,\qquad \eta(t_2)=0$
  • Entonces: $\dot q(t)\to \dot q(t)+\varepsilon \dot\eta(t)$
  • La variación de la acción queda:
  • $$\delta S=\int_{t_1}^{t_2}\left(\frac{\partial L}{\partial q}\eta+\frac{\partial L}{\partial \dot q}\dot\eta\right)dt$$

Integración por partes en el tiempo

Configuración
Integramos por partes el segundo término:
$$\int_{t_1}^{t_2}\frac{\partial L}{\partial \dot q}\dot\eta\,dt$$
Elegimos:
$$u=\frac{\partial L}{\partial \dot q}$$
$$dv=\dot\eta\,dt$$
Entonces:
$$du=\frac{d}{dt}\left(\frac{\partial L}{\partial \dot q}\right)dt$$
$$v=\eta$$
Resultado
$$\int_{t_1}^{t_2}\frac{\partial L}{\partial \dot q}\dot\eta\,dt$$
$$=\left[\frac{\partial L}{\partial \dot q}\eta\right]_{t_1}^{t_2}$$
$$-\int_{t_1}^{t_2}\frac{d}{dt}\left(\frac{\partial L}{\partial \dot q}\right)\eta\,dt$$
Como $\eta(t_1)=\eta(t_2)=0$, el borde desaparece.
Luego:
$$\int_{t_1}^{t_2}\frac{\partial L}{\partial \dot q}\dot\eta\,dt$$
$$=-\int_{t_1}^{t_2}\frac{d}{dt}\left(\frac{\partial L}{\partial \dot q}\right)\eta\,dt$$

Obtención de las ecuaciones de Lagrange

  • Entonces:
  • $$\delta S=\int_{t_1}^{t_2}\frac{\partial L}{\partial q}\eta\,dt-\int_{t_1}^{t_2}\frac{d}{dt}\left(\frac{\partial L}{\partial \dot q}\right)\eta\,dt$$
  • Reuniendo términos:
  • $$\delta S=\int_{t_1}^{t_2}\left[\frac{\partial L}{\partial q}-\frac{d}{dt}\left(\frac{\partial L}{\partial \dot q}\right)\right]\eta\,dt$$
  • Como $\eta(t)$ es arbitraria en el interior del intervalo, se obtiene:
  • $$\frac{d}{dt}\left(\frac{\partial L}{\partial \dot q}\right)-\frac{\partial L}{\partial q}=0$$

Generalización a varios grados de libertad

  • Si el sistema tiene coordenadas generalizadas $q_1,\dots,q_n$, entonces:
  • $$L=L(q_1,\dots,q_n,\dot q_1,\dots,\dot q_n,t)$$
  • y la acción es:
  • $$S=\int_{t_1}^{t_2}L\,dt$$
  • Variando cada coordenada de forma independiente, se obtiene para cada $i$:
  • $$\frac{d}{dt}\left(\frac{\partial L}{\partial \dot q_i}\right)-\frac{\partial L}{\partial q_i}=0,\qquad i=1,\dots,n$$

Ejemplo: Partícula libre

  • Una partícula libre tiene solo energía cinética:
  • $$L = T = \frac{1}{2}m\dot{\vec{r}}^2 = \frac{1}{2}m(\dot{x}^2 + \dot{y}^2 + \dot{z}^2)$$
  • Para la coordenada $x$:
  • $$\frac{\partial L}{\partial x} = 0$$
  • $$\frac{\partial L}{\partial \dot{x}} = m\dot{x}$$
  • Aplicamos Euler-Lagrange:
  • $$\frac{d}{dt}(m\dot{x}) - 0 = 0$$
  • $$m\ddot{x} = 0$$
  • La aceleración es cero, el momentum se conserva:
  • $$p_x = m\dot{x} = \text{constante}$$

Recuperación de las ecuaciones de Newton

  • Para un sistema conservativo en una dimensión:
  • $$L = T - U = \frac{1}{2}m\dot{x}^2 - U(x)$$
  • Calculamos las derivadas:
  • $$\frac{\partial L}{\partial x} = -\frac{dU}{dx} = -\frac{\partial U}{\partial x}$$
  • $$\frac{\partial L}{\partial \dot{x}} = m\dot{x}$$
  • Aplicamos Euler-Lagrange:
  • $$\frac{d}{dt}(m\dot{x}) - \left(-\frac{\partial U}{\partial x}\right) = 0$$
  • $$m\ddot{x} = -\frac{\partial U}{\partial x} = -\frac{dU}{dx}$$
  • Pero $F = -\frac{dU}{dx}$, luego:
  • $$m\ddot{x} = F$$
  • ¡Hemos recuperado la segunda ley de Newton!

Resumen final del bloque

  1. Se define una funcional.
  2. Se deforma la curva o trayectoria.
  3. Se calcula la variación a primer orden.
  4. Se integra por partes.
  5. Se usan extremos fijos.
  6. Se obtiene Euler–Lagrange.

En cálculo variacional ordinario:

$$\dfrac{d}{dx}\left(\dfrac{\partial \mathcal E}{\partial y'}\right)-\dfrac{\partial \mathcal E}{\partial y}=0$$

En mecánica:

$$\dfrac{d}{dt}\left(\dfrac{\partial L}{\partial \dot q_i}\right)-\dfrac{\partial L}{\partial q_i}=0$$

Ése es el puente entre variación y dinámica.

Cierre: Síntesis final

El cálculo variacional no es un adorno formal. Es el mecanismo que permite pasar de una condición global sobre trayectorias a una ecuación diferencial local.
